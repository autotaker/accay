# accay-interface-semantics

Use this skill to draft or revise `docs/acceptance/components/{component}/semantics.md`.

Read the component's public interface contracts first:

- `interfaces/openapi.yaml`
- `interfaces/operations.yaml`
- `interfaces/schemas/*.schema.json`

Then summarize:

- component responsibility
- operation meaning
- field semantics
- invariants
- forbidden transformations
- downstream meaning preservation
- common misunderstandings
